Coding Cerrado To-do List

React, TypeScript, JavaScript, HTML5 and SASS

This To-do List project was developed in collaboration with friends from the Coding Cerrado organization on GitHub. Our goal was to apply Scrum and Kanban methodologies, starting a project from scratch in React, taking it from a Figma design to a fully functional application.

Key tools and technologies used in this project include:

  • TypeScript for type safety and code clarity;
  • React Hooks for managing state and lifecycle events;
  • Local Storage for persistent data storage.

This project provided an excellent opportunity to practice agile methodologies and teamwork, while also honing our skills in modern web development technologies.
